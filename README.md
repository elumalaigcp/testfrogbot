JFrog Repository (Test Artifiacts Promotion)

## Project Description
This repository contains a Proof of Concept (PoC) for integrating JFrog products (Artifactory, Xray, Frogbot) with our application development. The goal of this PoC is to demonstrate the capabilities of JFrog Artifactory in managing and automating software artifacts, improving build and release processes.

### Product Description:

- **Artifactory** - It is a central hub for storing and managing software artifacts, container images, and configuration files. Similar to Sonatype Repository Manager
- **Xray** - It scans artifacts and dependencies for known security vulnerabilities, licensing compliance. Similar to Nexus IQ.
- **Frogbot** - It periodically scans repositories for security vulnerabilities, license compliance, secrets detection, and SAST scan. Similar to Sonatype's Continuous Monitoring.
